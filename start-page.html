üí° How to Use:

Type commands in the prompt below and press Enter:

  • go <url>: Navigate to a website (e.g., go example.com)
  • EALRY ACCESS: search <query>: Search the web (e.g., search latest tech news)
  • EALRY ACCESS: s <query>: Shortcut for search (e.g., s weather today)
  • dashboard: Go to your personalized dashboard
  • test <page-name>: Load a local HTML file from the 'test-pages' folder
  • list-tests: Display a list of all available local test pages
  • list-languages: List available language
  • config: View your configuration
  • config <option> <value>: Update a config
  • click [<id>]: Simulate clicking a link
  • back: Go back in Browse history
  • quit: Exit the browser
  • generate <tag name>: Generate a random test page
