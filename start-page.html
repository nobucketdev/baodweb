Welcome to Your Fancy Terminal Browser!

Navigate the web with style, right from your terminal. This browser is designed for efficiency and a unique visual experience.

Feature Description
Fast Rendering Experience quick, text-based display of web content.

Image Rendering View images converted into engaging ASCII/terminal art.
Local File Browsing Access and display local HTML files from the 'test-pages' directory.
Dynamic Widgets Display live time, weather, and news directly on your dashboard.

üí° How to Use:

Type commands in the prompt below and press Enter:

  • go <url>: Navigate to a website (e.g., go example.com or go https://news.ycombinator.com)
  • dashboard: Go to your personalized dashboard with live widgets.
  • test <page-name>: Load a local HTML file from the 'test-pages' folder (e.g., test features to load features.html. The .html extension is optional!)
  • list-tests: Display a list of all available local test pages in the 'test-pages' directory.
  • list-languages: List available language
  • config: A tool help you see you configuration
  • config <option> <value>: Config option, e.g., config language vi - change current language to Vietnamese
  • click <id show in bracket [] befor a anchor>: Emulating clicking a link and take you to the linked url
  • back: Go back to the previous page in your browsing history.
  • quit: Exit the browser application.
  • generate <tag name>: Generate a random test page using the given tag.
