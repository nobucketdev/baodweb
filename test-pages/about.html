About BaodWeb Terminal Browser

Your lightweight, text-based web browsing experience.

What is BaodWeb?

BaodWeb is a unique terminal-based web browser designed for efficiency and simplicity. It allows you to navigate the web, view content, and interact with information directly from your command line interface. Forget graphical clutter; BaodWeb brings the web to your text-only environment.

It's perfect for quick information retrieval, browsing on low-resource systems, or simply for those who prefer a minimalist approach to web interaction.

Key Features

  • Visual ANSI Rendering: Enhance the interface with colored and styled texts, boxs,...
  • Many HTML features: View and interact with HTML content, including images and links.
  • Image Display: View images directly in the terminal using Half block character
  • Paging and Scrolling: Easily navigate through long pages.
  • Link Navigation: Clickable links within the terminal (via IDs).
  • Configurable Settings: Customize your browsing experience (e.g., color, rendered tags).
  • Dashboard View: Get quick local time, weather, and news updates.
  • Local File Support: Open and view local HTML files.

How to Use

BaodWeb operates through simple commands entered in your terminal. Here are a few examples:

  • go <url>: Navigate to a website (e.g., go example.com)
  • dashboard: View your personalized dashboard.
  • up / down: Scroll through the current page.
  • click <id>: Follow a link by its displayed ID.
  • back: Go back in your browsing history.
  • config: View and modify browser settings.
  • help: Display a list of all available commands.
  • quit: Exit the browser.
