Text Formatting Tags

The browser now supports various inline text formatting tags:

  • Bold text using <b> or Strong text using <strong>.
  • Italic text using <i> or Emphasized text using <em>.
  • Underlined text using <u>.
  • Deleted text using <del>.
  • Inserted text using <ins>.
  • Highlighted text using <mark>.
  • Subscript text H2O using <sub>.
  • Superscript text E=MC2 using <sup>.
  • A span element for generic inline containers.

Image Rendering (ASCII/Terminal Art)

Images are converted into a low-resolution pixel art representation using ANSI half-block characters. This allows for a visual approximation of images without needing a graphical display.
