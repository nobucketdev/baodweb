Welcome to BaodWeb Terminal Browser!

Looks like this is your first time using BaodWeb. We're excited to have you!

BaodWeb is a simple text-based web browser designed for the terminal.

You can browse websites, view local HTML files, and even generate test pages.

To get started, try one of these commands:

  • go example.com - Visit a website
  • dashboard - See your personalized dashboard
  • list-tests - See available test pages
  • help - View all commands

Enjoy your browsing experience!
