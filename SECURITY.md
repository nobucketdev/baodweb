# Security Policy

## Supported Versions

BaodWeb is currently in early development. Only the latest release is actively maintained and eligible for security updates.

| Version       | Supported        |
|---------------|------------------|
| 1.2.0 - 1.2.4 | Yes              |
| 1.1.0 - 1.1.2 | Yes              |
| 1.0.0         | Yes              |
| < 1.0.0       | No               |
