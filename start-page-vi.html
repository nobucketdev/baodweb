🚀 Chào mừng đến với Trình Duyệt Terminal Đẹp Mắt của Bạn! 🚀

Duyệt web theo phong cách riêng, ngay trong terminal. Trình duyệt này được thiết kế để tối ưu hiệu suất và mang lại trải nghiệm hình ảnh độc đáo.

✨ Tính năng nổi bật:

Tính năng Mô tả
Hiển thị nhanh Trình bày nội dung web dưới dạng văn bản một cách nhanh chóng.
Hỗ trợ HTML cơ bản Hiển thị tiêu đề, đoạn văn, danh sách, liên kết và hơn thế nữa.
Hiển thị hình ảnh Xem hình ảnh được chuyển thành nghệ thuật ASCII/terminal hấp dẫn.
Duyệt tập tin cục bộ Truy cập và hiển thị các tệp HTML trong thư mục 'test-pages'.

💡 Cách sử dụng:

Nhập lệnh vào dòng dưới đây và nhấn Enter:

  • go <url>: Truy cập một website (ví dụ: go example.com hoặc go https://news.ycombinator.com)
  • test <tên-trang>: Tải tệp HTML cục bộ từ thư mục 'test-pages' (ví dụ: test features để tải features.html. Phần đuôi .html là tùy chọn!)
  • list-tests: Liệt kê tất cả các trang thử có sẵn trong thư mục 'test-pages'.
  • back: Quay lại trang trước trong lịch sử duyệt.
  • quit: Thoát khỏi trình duyệt.
  • generate <tên tag>: Tạo trang thử ngẫu nhiên với tag chỉ định.
